# 家事商談報告

一、主訴問題  
案主於士林法院家事服務中心接受家事商談，主要針對夫妻分居及子女探視安排進行討論。

二、個案概況  
(一)家庭  
1. 案主與陳先生於2010年認識，2015年結婚，育有兩名子女：匡匡（6歲）及亮亮（4歲）。  
2. 案主與陳先生於113年2、3月因陳先生外遇而分居，陳先生自此離家。  
3. 案主目前與父母及兩名子女同住，家庭支持系統良好。  
4. 案主的父母經營家族企業，案主在家中工作，並負責照顧子女。  
5. 案主表示陳先生在外遇後，對家庭的責任感減弱，並未提供經濟支持。  

(二)子女狀況  
1. 案主目前負責兩名子女的日常照顧，並在學校接送。  
2. 子女對於父親的探視表現出抗拒，尤其是匡匡，曾要求請假不想上學。  
3. 子女未曾主動詢問父母分居原因，案主推測他們可能已知曉部分情況。  
4. 案主擔心陳先生的行為對子女造成心理壓力，特別是在探視期間。  

(三)訴訟  
1. 案主與陳先生因離婚及親權問題進行法律訴訟，陳先生要求高額贍養費（1千萬）。  
2. 案主對於陳先生的要求表示無法接受，並認為其不合理。  
3. 案主已經參加心理諮商，並在法律上尋求支持以保護自己及子女的權益。  

三、社工評估  
案主在面對陳先生的外遇及隨之而來的家庭變故中，展現出堅強的應對能力。她積極參與心理諮商，並努力維持家庭的穩定。案主對於子女的情感需求十分關注，並希望能夠在法律上爭取合理的探視安排及扶養費。社工建議案主持續與法律顧問合作，並在情感上尋求支持，以便更好地應對未來的挑戰。
